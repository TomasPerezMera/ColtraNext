# ColtraNext

Modernización del E-Commerce "El Rincón de Coltrane" utilizando NextJs como motor.

Comando para levantar la aplicación:

```bash
npm run dev
```

Comando para Build/Deploy:

```bash
npm run build
```

Rutas Disponibles:

```
Route (app)
┌ ○ /
├ ○ /_not-found
├ ○ /admin/products
├ ƒ /admin/products/[id]
├ ƒ /api/products
├ ƒ /api/products/[id]
├ ○ /cart
├ ○ /checkout
├ ○ /login
├ ○ /products
├ ƒ /products/[slug]
├ ○ /profile
└ ○ /register
```
